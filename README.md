# Sistema de Gerenciamento de Clientes

Projeto desenvolvido em Python com SQLite para praticar lógica de programação, manipulação de banco de dados e operações CRUD.

## Funcionalidades
- Adicionar clientes
- Listar clientes cadastrados
- Atualizar informações
- Remover clientes
- Validar nome, email e ID
- Menu interativo no terminal

## Tecnologias utilizadas
- Python
- SQLite
- SQL

## Como executar

1. Abra a pasta do projeto no VS Code
2. Execute o arquivo:

```bash
py app.py
